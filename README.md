# Template for Svelte Web Components

https://github.com/dariuszsikorski/svelte-webcomponent

Most of the code is from the template and then some adjustments were made to fit our use case.
We will be shipping a standalone web component that will be displayed in an existing app.

## Quick Start

### Installation

```bash
pnpm install
```

### Development

```bash
pnpm run dev
```

### Production Build

```bash
pnpm run build
```

### Preview Production Build

```bash
pnpm run standalone
```
